Cara Bergabung

Pilih platform aplikasi/game yang ingin kamu gunakan:

Tutorial Host Go
  1. Unduh dan Instal (Tanpa Root) HostGo APK
  2. Buka Host GO dan klik "Editor Host".
  3. Klik kotak centang di sebelah kanan, dan Klik "Unduh Berkas Host".
  4. Ketik tautan: https://download.gtpshost.com/TechTopia
  5. Klik "Unduh dan Terapkan".
  6. Kembali dan klik "MULAI".
  7. Buka Growtopia dan Hubungkan!
CATATAN: Jika Anda mengalami masalah saat masuk, silakan buka link komunitas resmi TechTopia GTPS.
Tutorial Virtual Host
  1. Unduh dan Instal: HostGo APK
  2. Unduh Host: Unduh Host Disini
  3. Setelah terinstal, buka Virtual Host dan klik "SELECT HOSTS FILE".
  4. Temukan file Host dan klik file tersebut.
  5. Buka Growtopia dan Hubungkan!
CATATAN: Jika Anda mengalami masalah saat masuk, silakan buka link komunitas resmi TechTopia GTPS.
Tutorial POWERTUNNEL
  1. Unduh dan Instal: POWERTUNNEL APK
  2. Buka tombol Plugin di samping tombol Pengaturan.
  3. Temukan Host dan buka Pengaturan.
  4. Ketik tautan: https://growtopia.id/android
  5. Kembali dan klik tombol "CONNECT".
  6. Buka Growtopia dan Hubungkan!
CATATAN: Jika Anda mengalami masalah saat masuk, silakan buka link komunitas resmi TechTopia GTPS.
Tutorial WINDOWS
  1. Tekan Win+R (Tombol Windows+R).
  2. Ketik C:\Windows\System32\drivers\etc di kotak teks.
  3. Klik kanan pada file "hosts" dan edit dengan Notepad/editor teks.
  4. Tambahkan ip berikut:
    157.230.42.146 growtopia1.com
    157.230.42.146 growtopia2.com
    157.230.42.146 www.growtopia1.com
    157.230.42.146 www.growtopia2.com
    157.230.42.146 RvLnd.here
  5. Simpan filenya.
  6. Buka Growtopia dan sambungkan!
CATATAN: Jika Anda mengalami masalah saat masuk, silakan buka link komunitas resmi TechTopia GTPS.
Tutorial Surge5
  1. Ikuti langkah pada platform Android, gunakan aplikasi VPN pilihan Anda.
  2. Unduh dan Instal: Surge5 APK
  3. Buka lalu Tekan "OK".
  4. Klik "Default.conf".
  5. Masukkan url https://ios.gtpshost.com/TechTopia
  6. Klik "OK" dan "Selesai"
  7. Tekan "SETUP" lalu setujui kebijakan dengan mengeklik "OK" dan "Izinkan" untuk Konfigurasi VPN!.
  8. Selesai, lalu buka "Growtopia" dan Hubungkan!
CATATAN: Jika Anda mengalami masalah saat masuk, silakan buka link komunitas resmi TechTopia GTPS.

Tentang Server

TechTopia GTPS | Server Pribadi Growtopia

Selamat datang di TechTopia GTPS.
Didirikan pada 10 Agustus 2025 oleh KayNest1 & RentaGT, TechTopia GTPS telah berkembang menjadi komunitas yang berkembang pesat dengan lebih dari 50+ anggota. Lebih dari sekadar permainan, kami adalah wadah kreativitas dan kolaborasi bagi para gamer Indonesia.

Pendiri: KayNest1 & RentaGT
Tanggal Berdiri: 10 Agustus 2025

FAQ & Bantuan
Bagaimana cara bergabung ke server TechTopia GTPS? ▶
Pilih platform di bagian "Cara Bergabung", lalu ikuti tutorial sesuai aplikasi yang Anda gunakan.
Bagaimana cara mengikuti event? ▶
Pantau informasi event terbaru di website dan grup WhatsApp. Ikuti instruksi yang diberikan setiap event berlangsung.
Siapa saja tim pengelola TechTopia GTPS? ▶
Tim terdiri dari Owner, Co-Owner, dan beberapa moderator. Info lengkap dapat ditemukan di bagian "Tentang Server".
Bagaimana cara mendapatkan bantuan jika mengalami masalah? ▶
Silakan hubungi kami via email, WhatsApp, atau Instagram di bagian "Komunitas & Kontak".
